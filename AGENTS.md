# Repository Agent Guide

## Purpose

This repository is a focused C++ interview training workspace for a candidate returning to modern C++ after a long professional gap.

## Session Goals

1. Keep preparation practical and interview-focused.
2. Track progress day by day across an 8-day plan.
3. Emphasize C++20 fundamentals, STL/ranges, and coding interview patterns.

## Files To Update Every Session

1. `docs/progress/overall.md`
2. One or more files under `docs/progress/sections/`

## Working Rules For AI Agent

1. Start each session by reviewing pending items in progress files.
2. Propose concrete drills (problems, constraints, target complexity).
3. Record what was completed and what needs reinforcement.
4. Keep examples C++20-compatible and compiler-friendly for MSVC.
